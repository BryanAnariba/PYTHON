1. Siempre creamos el entorno virtual
  python -m venv .venv
  .\.venv\Scripts\activate

2. Instalamos dependencias
  pip install djangorestframework

3. Creamos proyecto y nos ubicamos dentro de el
  django-admin startproject ecommerce_backend

4. Ver los archivos de la carpeta settings yo los cree, basicamente es pasar lo de settings.py de ecommerce_backend a esos tres ficheros en settings y crea el archivo __init__.py

5. Crea la carpeta apps con el archivo __init__.py, esta contendra todos los modulos o aplicaciones, users, categories, products, contabilida, caja etc, y como los crearas
  con el comando
    django-admin startapp nombre_modulo
